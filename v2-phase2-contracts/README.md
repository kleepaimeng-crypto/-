# 乘客实时动态开发契约

本目录只描述主导航“乘客实时动态”界面。开发前依次阅读 `SPEC.md`、`API.md`、`schema.md`、`SLICES.md` 和 `AGENTS.md`。

界面基线：

- 飞机底图：`docs/CA_332-1 2.svg`。
- 编号座位图层：`docs/plane-seat-overlay-renamed.svg`。
- 影音排行参考：`docs/图6更改补充.png`。
- 整体布局参考：`docs/前中后舱网联数据平台 6.png`。

本期只实现影音类型排行、可滚动舱位图、116 个舷窗透光度、乘客观看/浏览详情及座位双向定位。驾驶舱实时监控只保留静态面板，不开发接口。
