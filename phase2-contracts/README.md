# 乘客实时动态开发契约

本目录是“乘客实时动态”单界面的开发基线，供所有参与者共同使用。开始开发前依次阅读：

1. `SPEC.md`：页面行为与验收标准。
2. `API.md`：该页面专用的只读接口。
3. `schema.md`：现有数据表及统计口径。
4. `SLICES.md`：并行切片和依赖。
5. `AGENTS.md`：本切片实施约束。

需求来源优先级：用户本次补充要求 > `docs/图6更改补充.png` > `docs/前中后舱网联数据平台 6.png` > `docs/前中后舱网联数据分析平台.docx`。

本期只开发主导航“乘客实时动态”页面，不开发“数据统计”页面，不提供流量、吞吐、带宽或丢包接口。左上角乘客影音区域按模拟器接收端的类型出现次数展示，不显示 Mbps。
